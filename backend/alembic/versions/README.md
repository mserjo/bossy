# Версії Міграцій Alembic

Ця директорія містить окремі скрипти міграцій, згенеровані Alembic.
Кожен файл представляє набір змін до схеми бази даних.

Не редагуйте файли в цій директорії вручну, якщо ви не дуже добре знайомі з Alembic та його системою версіонування.
Нові скрипти міграцій зазвичай повинні генеруватися за допомогою команд Alembic, наприклад:

```sh
alembic revision -m "опишіть_вашу_зміну"
```
(Замініть `"опишіть_вашу_зміну"` на коротке, але інформативне повідомлення про суть міграції, наприклад, `"add_user_table"` або `"add_email_to_users_table"`).

А потім наповнюватися необхідними операціями в функціях `upgrade()` та `downgrade()`.
