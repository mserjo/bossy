# backend/app/src/locales/README.md
# # Директорія Файлів Локалізації (Перекладів) на Основі JSON
#
# Ця директорія містить файли перекладів для інтернаціоналізації (i18n)
# та локалізації (l10n) програми Kudos. Переклади зберігаються у форматі JSON.
#
# ## Структура Директорії
#
# Кожна підтримувана мова має власну піддиректорію, названу відповідно до
# коду мови (наприклад, `en` для англійської, `uk` для української).
#
# Всередині кожної мовної директорії знаходиться файл `messages.json`.
# Цей файл містить пари ключ-значення, де ключ є оригінальним рядком
# (зазвичай англійською) або унікальним ідентифікатором рядка, а значення –
# його перекладом на відповідну мову.
#
# -   `en/messages.json`: Файл перекладів для англійської мови.
# -   `uk/messages.json`: Файл перекладів для української мови.
#
# ## Формат `messages.json`
#
# Файли `messages.json` є простими JSON-об'єктами:
#
# ```json
# {
#   "hello_world": "Hello, World!",
#   "welcome_message": "Welcome to the Kudos application, {username}!",
#   "errors": {
#     "not_found": "The requested item was not found.",
#     "permission_denied": "You do not have permission to perform this action."
#   }
# }
# ```
# (У відповідному файлі `uk/messages.json` будуть українські переклади цих ключів.)
#
# ## Використання в Коді
#
# Модуль `backend/app/src/core/i18n.py` відповідає за завантаження цих JSON-файлів
# та надання функції для отримання перекладених рядків за ключем для поточної
# вибраної локалі.
#
# ## Додавання та Оновлення Перекладів
#
# 1.  **Визначте ключ**: Для нового рядка, що потребує перекладу, придумайте унікальний
#     ключ (наприклад, `user_profile_title`) або використовуйте базовий англійський текст як ключ.
# 2.  **Додайте ключ до `en/messages.json`**: Внесіть ключ та його англійське значення.
# 3.  **Додайте ключ та переклад до `uk/messages.json`**: Внесіть той самий ключ та його
#     український переклад.
# 4.  **Використовуйте функцію перекладу**: В коді програми використовуйте функцію з `i18n.py`
#     (наприклад, `_("user_profile_title")`) для відображення перекладеного тексту.
#
# ## Примітки
#
# - На даний момент файли `messages.json` ініціалізовані як порожні (`{}`).
#   Вони будуть наповнюватися по мірі розробки та виявлення рядків, що потребують перекладу.
# - Розгляньте можливість використання інструментів або скриптів для автоматизації
#   збору ключів з коду або для перевірки повноти перекладів, якщо проект значно розростеться.
