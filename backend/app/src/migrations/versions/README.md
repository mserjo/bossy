# Alembic Migration Versions

This directory contains the individual version migration scripts generated by Alembic.
Each file represents a set of changes to the database schema.

Do not manually edit files in this directory unless you are very familiar with Alembic
and its versioning system. New migration scripts should typically be generated using
Alembic commands like:

```sh
alembic revision -m "describe_your_change"
```

And then populated with the necessary `upgrade()` and `downgrade()` operations.
