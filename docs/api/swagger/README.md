# Swagger UI / OpenAPI Документація

Інтерактивна документація API, згенерована на основі OpenAPI специфікації,
зазвичай доступна за такими шляхами в запущеному backend додатку:
- `/docs` - Swagger UI
- `/redoc` - ReDoc UI

Специфікація OpenAPI (`openapi.json`) генерується автоматично і може бути знайдена [тут](../openapi.json).
