# Архітектурні Діаграми

В цьому каталозі зберігаються візуальні діаграми, що ілюструють різні аспекти архітектури системи:
- Компонентна діаграма
- ER діаграма бази даних
- Діаграми бізнес-процесів (BPMN)
- Діаграма розгортання
- Діаграма стеку технологій

Файли діаграм (наприклад, у форматі PlantUML `.puml` або BPMN `.bpmn`) будуть додані сюди.
